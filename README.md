# 📌 Project Overview

This project is about being a Student Digital Ambassador. The main goal of a Student Digital Ambassador is to help students become more comfortable with technology and the internet. A Student Digital Ambassador wants to teach students skills like creating a good online profile learning to code working with others online sending proper emails and being safe on the internet.

The project is divided into five tasks. Each task looks at a part of what it means to be digitally literate. All of the tasks that a Student Digital Ambassador has to do are written down here along with files and thoughts, about what was learned.


# 📂 Module Summaries

## 🔹 module 1 - Introduction to Digital Literacy

This module introduced the concept of digital literacy and its importance. I explored tools like Cooltext, Prezi, Miro, Canva, and GIFs to create engaging digital content. Using Canva, I designed an infographic explaining key aspects of digital literacy.

## 🔹  Module 2 – Building a Digital Portfolio
This module focused on creating a professional online presence using platforms like LinkedIn, GitHub, Kaggle, Quora, ResearchGate, and TechGig. I created profiles on GitHub, LinkedIn, and Kaggle to showcase my skills, connect with professionals, and explore opportunities in AI and ML.


## 🔹 module 3 - Programming & Collaboration Platforms
In this module, I learned about coding platforms like HackerRank, HackerEarth, CodeChef, and CodeTantra, along with collaboration tools in Google Workspace such as Google Docs, Sheets, Slides, and Forms. I completed a Python challenge on HackerRank and created a Google Form quiz.

## 🔹 module 4 - Email & Social Media Etiquette
This module emphasized the importance of professional communication. It covered email structure, guidelines, and workplace etiquette. I applied these concepts by drafting professional emails and creating a checklist for responsible social media usage.

## 🔹 module 5 - Cybercrime Awareness
This module focused on different types of cybercrimes, including financial fraud and personal data theft. It also covered prevention measures and awareness strategies. I created a case study and a checklist to help students stay safe online.
